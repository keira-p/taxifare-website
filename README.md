# NYC Taxi Fare Predictor 🚕

This project is a full-stack ML system that predicts NYC taxi fares.

## Architecture

- FastAPI backend deployed on Google Cloud Run
- Trained ML model served via REST API
- Streamlit frontend deployed on Streamlit Cloud
- Frontend calls backend via HTTP requests

## Features

- Interactive ride input (datetime, coordinates, passengers)
- Map visualisation of journey
- Real-time fare prediction via API
- Error handling and validation

## Tech Stack

- Python
- FastAPI
- scikit-learn
- Streamlit
- Google Cloud Run
- requests
